Overview of exMarker api

ExMarker provides JSON-based web service APIs for fetching bulk results. You can use the APIs by any language that can handle HTTP request and response.

ExMarker API Content
Name URL Content HTTP Method
Browse DNA results http://exmarker.zhangshang.online/exmarker/php/browse_dna.php JSON contents of the Browse DNA results GET
